# Reports Directory

This directory contains **auto-generated SOC reports** created by Mini-SOC.

## Report Files
Each run may generate:
- `alerts_<timestamp>.json`

## Report Contents
- Executive summary
- Total alert count
- Severity breakdown
- Individual alert details

## Purpose
Used for:
- SOC analyst review
- Incident documentation
- SIEM ingestion (future)

⚠️ Files are generated automatically. Do not edit manually.
